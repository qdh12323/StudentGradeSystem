## 🎯 项目概述

本系统是一个完整的学生信息管理系统，以综合测评为核心，扩展了学生信息管理和课程信息管理功能。系统基于真实的2024级数据科学与大数据技术2班综测数据开发，支持26名真实学生的完整信息管理。

## 🏗️ 数据库设计核心

### 主要实体 (7个核心表)

1. **Students** - 学生信息表 (支持BIGINT学号)
2. **Classes** - 班级信息表
3. **Courses** - 课程信息表
4. **ComprehensiveEvaluations** - 综合测评主表
5. **BonusDetails** - 加分详情表
6. **CourseOfferings** - 课程开设表
7. **Users** - 用户管理表

### 关系设计

```
班级 (1:N) 学生 (1:N) 综合测评 (1:N) 加分详情
课程 (1:N) 课程开设
用户 (1:1) 学生
```

### 评分体系

```
总积分(P) = 学业成绩(X) + 创新实践(C) + 社会实践(S) + 文体实践(W)

创新实践(C) = C1(基本分) + C2(加分)
社会实践(S) = S1(学生工作) + S2(社会服务) + S3(奖励加分)
```

## 🔧 技术实现特色

### 数据库层面
