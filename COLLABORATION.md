## 🌳 分支策略

### 分支说明
- `main`: 主分支，稳定版本
- `dev-qdh`: qdh的开发分支
- `dev-同学名字`: 同学的开发分支

### 工作流程

#### 1. 初始设置（同学第一次参与）
```bash
# 克隆仓库
git clone git@github.com:qdh12323/StudentGradeSystem.git
cd StudentGradeSystem

# 创建自己的开发分支
git checkout -b dev-你的名字
git push -u origin dev-你的名字
```

#### 2. 日常开发流程
```bash
# 开始工作前，同步最新代码
git checkout dev-你的名字
git pull origin main
git merge main

# 开发功能
# ... 编写代码 ...

# 提交代码
git add .
git commit -m "feat: 添加登录页面"
git push origin dev-你的名字
```

#### 3. 合并到主分支
1. 在GitHub上创建Pull Request
2. 选择：`dev-你的名字` -> `main`
3. 添加描述，@对方进行代码审查
4. 审查通过后合并

